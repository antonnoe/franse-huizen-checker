# franse-huizen-checker

Welkom!
Deze repository is bedoeld voor het analyseren, vergelijken en checken van Franse huizen (vastgoed).
Hier kun je Python- of ander code plaatsen om automatische checks uit te voeren, data te verzamelen en analyseren.

## Gebruik
- Voeg je eigen scripts toe in deze repo.
- Schrijf uitleg en opmerkingen hier in de README.
